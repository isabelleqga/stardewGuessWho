## :pencil: Descrição
Aplicação simples utilizando HTML, CSS e JavaScript básicos para simular um jogo de "cara-a-cara" com os personagens do jogo Stardew Valley. Utilização local em máquinas diferentes.

## :evergreen_tree: Personagens
- Abigail, Emily, Haley, Leah, Maru, Penny
- Alex, Elliott, Harvey, Sam, Sebastian, Shane
- Caroline, Clint, Demetrius, Evelyn, George, Gus, Jas, Jodi, Kent, Leo, Lewis, Linus, Marnie, Pam, Pierre, Robin, Vincent, Willy
- Dwarf, Gunther, Henchman, Krobus, Morris, Wizard

## :sunflower: Como jogar
- Abra a aplicação do jogo para ver o tabuleiro;
- Faça a sua escolha de personagem com o qual quer jogar, digite seu nome no campo de tempo e clique em "pick";
- Com o seu personagem escolhido, faça perguntas objetivas (sim ou não) ao seu oponente para descobrir qual o personagem que ele escolheu;
- Clique na foto dos personagens que não condizem com as respostas do seu oponente;
- A última foto no seu tabuleiro que não estiver cinza deve ser o personagem que o seu oponente escolheu.
